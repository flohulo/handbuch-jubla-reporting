# Nutzung für Leitende

Diese Seite erklärt, wie du nach einer Gruppenstunde einen Bericht einreichst.

---

## Bericht einreichen

1. Öffne die App-URL deiner Schar im Browser (am besten als **Lesezeichen** auf dem Homescreen speichern)
2. Melde dich mit deinen Zugangsdaten an
3. Tippe auf **«Bericht erfassen»**
4. Fülle das Formular aus:
   - Datum der Gruppenstunde
   - Deine Gruppe
   - Anzahl Teilnehmende
   - Thema / Aktivität
   - Verlauf (kurze Beschreibung)
   - Besondere Vorkommnisse (falls vorhanden)
5. Tippe auf **«Absenden»**

!!! tip "Direkt nach dem Höck"
    Am besten reichst du den Bericht direkt im Anschluss an die Gruppenstunde ein – im Bus oder noch vor Ort. So vergisst du keine Details.

---

## Strike erfassen

Wenn ein Teilnehmender eine Verwarnung erhält:

1. Tippe auf **«Strike erfassen»**
2. Wähle den Namen der Person
3. Beschreibe den Vorfall kurz
4. Bestätige mit **«Speichern»**

!!! info "Strike ≠ Alarm"
    Ein Strike wird intern gespeichert, löst aber **keinen automatischen Alarm** aus. Nur das Ankreuzen von «Besonderer Vorfall – Alarm» im Bericht-Formular löst eine Alarm-Mail aus.

---

## App als Homescreen-Shortcut speichern (empfohlen)

=== "iPhone (Safari)"
    1. Öffne die App-URL in Safari
    2. Tippe auf das Teilen-Symbol ↑
    3. Wähle **«Zum Home-Bildschirm»**

=== "Android (Chrome)"
    1. Öffne die App-URL in Chrome
    2. Tippe auf das Menü ⋮
    3. Wähle **«Zum Startbildschirm hinzufügen»**
